1. 适配安卓12~16API，修补新版系统VPN唤起问题
2. 优化UI界面，优化代码效率
3. 添加自适应黑暗模式，适配导航栏状态栏沉浸
4. 适配多语言，支持简体中文、繁体中文（TW）、繁体中文（HK）、英文、日文。自适应系统语言设定
5. 修改了图标
